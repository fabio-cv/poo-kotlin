# Sistema de Controlo de Acesso e Permissões

Projeto em Kotlin que simula um sistema de segurança baseado em cargos e permissões,
aplicando conceitos de Programação Orientada a Objetos.

## Conceitos aplicados
- Encapsulamento
- Composição
- Enum classes
- Collections (Set e List)

## Funcionalidades
- Registo de utilizadores com senha cifrada
- Autenticação
- Atribuição de cargos
- Verificação de permissões

## Execução
Executar a função `main` no ficheiro `Main.kt`.
