# Biblioteca Digital

Sistema em Kotlin para gerenciar livros, filmes e músicas aplicando conceitos de POO.

## Funcionalidades
- Adicionar mídias
- Listar detalhes completos (polimorfismo)
- Buscar mídia por título
- Tentar reproduzir mídias (interface)

## Como executar
1. Abrir o projeto no IntelliJ
2. Rodar `Main.kt`
